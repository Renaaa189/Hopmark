## <img src="https://media.giphy.com/media/juua9i2c2fA0AIp2iq/giphy.gif" width="25"> Introducción

**Hopmark** es un sistema de gestión académica desarrollado en JavaScript para administrar estudiantes, carreras universitarias, materias y calificaciones dentro de una única plataforma.

El proyecto busca garantizar la coherencia del recorrido académico de cada estudiante mediante un conjunto de validaciones que controlan la relación entre carrera, año cursado e historial de materias. De esta forma, el sistema puede detectar inconsistencias académicas y mantener registros estructurados y confiables.

Además, incorpora un sistema visual de progreso basado en conejos, los cuales son asignados automáticamente según el rendimiento obtenido en cada materia.

Actualmente el sistema contempla distintas carreras universitarias, cada una con su propio plan de estudios, materias y estructura académica.

---

## Características principales

### Gestión académica

* Registro y administración de estudiantes
* Gestión de carreras universitarias
* Organización de materias por año académico
* Consulta del historial académico completo

### Validaciones del sistema

* Verificación de materias correspondientes a cada carrera
* Control de progresión académica entre años
* Detección de inconsistencias en el historial
* Validación automática de registros académicos

### Seguimiento del rendimiento

* Registro de calificaciones por materia
* Historial de desempeño académico
* Visualización estructurada de resultados
* Sistema de progresión basado en conejos

### Experiencia de usuario

* Interfaz moderna y organizada
* Navegación intuitiva
* Gestión centralizada de información académica
* Visualización clara de datos y estadísticas

##  Instalación del proyecto

<p>Para utilizar Hopmark necesitás:</p>

<ul>
  <li>Node.js 18 o superior</li>
  <li>MySQL</li>
  <li>Git</li>
</ul>

<pre><code>node --version</code></pre>

### Paso 1: Clonar el repositorio

<pre><code>git clone https://github.com/TU_USUARIO/Hopmark.git
cd Hopmark</code></pre>

### Paso 2: Instalar dependencias

#### Backend

<pre><code>cd api
npm install</code></pre>

#### Frontend

<pre><code>cd client
npm install</code></pre>

### Paso 3: Configurar la base de datos

<p>Crear una base de datos MySQL y configurar las credenciales correspondientes.</p>

### Paso 4: Ejecutar el backend

<pre><code>cd api
npm run dev</code></pre>

### Paso 5: Ejecutar el frontend

<pre><code>cd client
npm run dev</code></pre>

<p>Abrir en el navegador:</p>

<pre><code>http://localhost:5173</code></pre>
